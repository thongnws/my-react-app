# vite-react-app

## Install & Start

> **Compatibility Note:**
> Vite requires [Node.js](https://nodejs.org/en/) version 18+. However, some templates require a higher Node.js version to work, please upgrade if your package manager warns about it.

With NPM:

```bash
$ npm i
$ npm run dev
```

With Yarn:

```bash
$ yarn
$ yarn dev
```

With PNPM:

```bash
$ pnpm i
$ pnpm dev
```
